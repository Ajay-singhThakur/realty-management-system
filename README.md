RealtyOS is a full-stack (MERN) property management platform designed for luxury real estate agencies. It features a modern, animated landing page for clients and a powerful, sleek admin dashboard for managing property listings, client testimonials, and business leads.



## üöÄ Features

### **Client-Side (Landing Page)**
- **Modern UI/UX:** Built with Glassmorphism and Parallax scrolling effects.
- **Scroll Animations:** Sections reveal smoothly as the user explores.
- **Project Showcase:** Fetches live architectural data from the backend.
- **Testimonial Grid:** Displays professional client feedback via a card-based system.
- **Lead Capture:** Integrated contact form and newsletter subscription.

### **Admin-Side (Dashboard)**
- **Real-time Analytics:** Track total projects, leads, and subscribers via stat cards.
- **Content Builder:** Easy-to-use forms for uploading new projects and clients with image handling.
- **Live Preview:** Real-time rendering of project cards before publishing.
- **Lead Management:** Centralized view for all contact inquiries.
- **Audience Intelligence:** Manage and view newsletter subscribers.
